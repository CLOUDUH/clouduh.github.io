# 群晖查看硬盘 SMART 信息

## 操作方法

查看硬盘编号

```bash
fdisk -l
```

![](../attachment/Pasted%20image%2020250912151531.png)

SATA:

```bash
sudo smartctl -a -d sat /dev/sata1
```

NVME: 

```
sudo nvme smart-log /dev/nvme0
```
